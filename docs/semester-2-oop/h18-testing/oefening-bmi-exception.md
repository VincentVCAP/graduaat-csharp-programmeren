# Oefening BMI exception

Vul de code van `BMICalculator` aan door te testen op een gewicht minder dan 30. Werp dan een `ArgumentOutOfRangeException`.

Maak dan 2 testmethodes:

* **`TestBMIOutOfRange`**: gebruik een `ExpectedException`
* **`TestBMIOutOfRangeTryCatch`**: test hier de berekening van een bmi met gewicht 20 en gebruik `Try` - `Catch`
