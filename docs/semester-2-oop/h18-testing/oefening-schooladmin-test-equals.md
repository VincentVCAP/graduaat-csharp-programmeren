# Oefening SchoolAdmin test equals

We hebben in de klasse `Person` en de klasse `Course` een (override) methode `Equals` gemaakt . Bekijk de code die we daar geschreven hebben. \
Deze code hebben we nog niet gebruikt. We zullen door middel van testing nagaan of deze code goed werkt.

Maak een nieuwe (public) klasse in `TestSchoolAdmin` met de naam **`TestEquals`**. 

![](/img/img-3244.png)


Maak hier een testmethode **`TestEqualStudent`**:\
Hierin maak je 2 student-objecten met dezelfde naam en geboortedatum. Test of deze niet gelijk zijn aan elkaar volgens de `Equals`-methode van Student.

Maak hier een testmethode **`TestEqualLecturer`**:\
Hierin maak je 2 lectoren-objecten met dezelfde naam, geboortedatum en een lege takenlijst. Test of deze niet gelijk zijn aan elkaar volgens de `Equals`-methode van `Lecturer`.
