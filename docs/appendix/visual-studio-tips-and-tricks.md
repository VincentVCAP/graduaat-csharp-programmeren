# Visual Studio Tips & Tricks

## Snippets

Om sneller code te kunnen schrijven in Visual Studio, kan je gebruik maken van zogenaamde Snippets. In de les gebruiken we regelmatig de snippets voor *full properties* `propfull` en *Console.WriteLine()* `cw`

Een lijst van beschikbare snippets kan je hier vinden:

https://docs.microsoft.com/en-us/visualstudio/ide/visual-csharp-code-snippets?view=vs-2019
